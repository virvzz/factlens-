Приватность: текст, который вы проверяете, отправляется только на выбранный вами endpoint. Расширение не использует аналитику, трекеры и скрытые серверы. История проверок не сохраняется.

Настройки (включая API key) хранятся в browser.storage.local — это не полноценный менеджер секретов: другие расширения его не видят, но данные лежат на диске в профиле Firefox без шифрования.

Автоматический фактчекинг может ошибаться. Не используйте результат как окончательный источник истины.

Провайдер

Эта модель считается устаревающей у DeepSeek. Она может работать, но рекомендуется перейти на актуальную модель.

Переменные: {{model}}, {{system_prompt}}, {{user_prompt}}, {{max_tokens}}, {{temperature}}, {{language}}, {{strictness}}. Строковые переменные указывайте в кавычках: "{{user_prompt}}" — значения экранируются автоматически.

Авторизация

Сюда можно добавить provider-specific параметры, например reasoning/thinking-опции DeepSeek. Поля добавляются в тело запроса.

Параметры генерации

Распознавание речи (STT) — для проверки звука видео

Отдельный endpoint для превращения звука в текст (Whisper-совместимый API, обычно /v1/audio/transcriptions). Claude/DeepSeek звук не распознают — нужен STT-провайдер: OpenAI, Groq (есть бесплатный лимит) или локальный Whisper-сервер.
